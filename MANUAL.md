# BlueMark ユーザー説明書

**BlueMark** は、STEP / STP 形式の CAD ファイルをブラウザ上で表示・計測・注釈できる 3D ビューアです。

## 1. 画面構成

```
┌──────────────────────────────────────────────────────────┐
│  ヘッダー：アプリ名 / ファイル名 / 「ファイルを開く」ボタン │
├──────────────────────────────────────────────────────────┤
│                                                          │
│                                         ┌────────────┐   │
│           3D ビューポート               │  ツールバー │   │
│                                         └────────────┘   │
│                                                          │
│                               [ViewCube]  [全体表示]     │
└──────────────────────────────────────────────────────────┘
```

| 領域 | 説明 |
|------|------|
| ヘッダー | ファイルを開く操作・読み込み済みファイル名を表示 |
| 3D ビューポート | CAD モデルを表示・操作する主要エリア |
| ツールバー | 操作モード・表示モードの切り替えアイコン |
| ViewCube | 右下に表示されるカメラ方向インジケーター |

## 2. ファイルの読み込み

3 種類の方法でファイルを読み込めます。

### 方法 A：ドラッグ＆ドロップ

ファイル未選択時に表示されるドロップゾーンに、エクスプローラーから `.step` / `.stp` ファイルをドラッグしてください。

### 方法 B：ファイルピッカー

ヘッダー右端の **「ファイルを開く」** ボタンをクリックするか、ドロップゾーンをクリックするとファイル選択ダイアログが開きます。

### 方法 C：postMessage（組み込み用）

iFrame として親アプリに埋め込んだ場合、JavaScript の `postMessage` でファイルデータを送ることができます。  
→ 詳細は [11. iFrame 組み込み / postMessage API](#11-iframe-組み込み--postmessage-api) を参照

> **対応形式：** `.step` / `.stp` のみ（ISO 10303 STEP ファイル）

## 3. マウス操作

| 操作 | 動作 |
|------|------|
| 左ドラッグ | モデルを回転（ピボット中心） |
| 右ドラッグ | パン（平行移動） |
| ホイール | ズームイン / ズームアウト（カーソル位置を中心に拡縮） |

## 4. ツールバー

ビューポート右側のツールバーには、左グループ（操作モード）と右グループ（表示モード）の 2 種類があります。

### 操作モード（左グループ）

| アイコン | モード | 説明 |
|----------|--------|------|
| カーソル | **ナビゲート** | 通常の回転・パン・ズーム（デフォルト） |
| 寸法線 | **寸法測定** | 2 点をクリックして距離を計測・3D 表示 |
| 吹き出し | **コメント** | クリック位置に 3D ピンコメントを配置 |
| 穴マーク | **穴検出** | メッシュを解析して穴・円筒を自動検出・表示 |

### 表示モード（右グループ）

| アイコン | モード | 説明 |
|----------|--------|------|
| 塗り潰し正方形 | **シェード** | シェード表示のみ（エッジなし） |
| 枠付き正方形 | **＋エッジ** | シェード＋全エッジ（デフォルト） |
| 破線グリッド | **陰線** | 背面のエッジを背景色でマスク |
| 格子 | **ワイヤー** | ワイヤーフレームのみ |

## 5. ViewCube（方向ナビゲーター）

ビューポート右下に表示される 3D キューブで、現在のカメラ方向を確認・操作できます。

| 操作 | 動作 |
|------|------|
| キューブの **面** をクリック | 上 / 下 / 正面 / 背面 / 右 / 左 のいずれかの標準ビューへスナップ |
| **▲ / ▼ / ◀ / ▶ 矢印** をクリック | 現在のカメラ方向から相対 90° 回転（連続クリックで一周） |
| **全体表示ボタン**（ViewCube 下）をクリック | 等角投影ビュー（斜め 45° 方向）にリセット、モデル全体にフィット |

> スナップ・リセット時はモデルのバウンディングボックスを基準に自動フィットします（350ms アニメーション）。

## 6. 寸法測定

ツールバーの **寸法測定モード** に切り替えて、2 点をクリックすることで距離を計測できます。

### 計測手順

1. ツールバーの「寸法線」アイコンをクリックして寸法測定モードに切り替え
2. モデルの **1 点目**（エッジ / 頂点 / 面）をクリック
3. **2 点目** をクリック → 寸法線と数値が 3D 上に表示される

### 寸法線の操作

| 操作 | 動作 |
|------|------|
| **矢印**をドラッグ | 寸法線全体（引き出し線＋ラベル）を移動。ドラッグ方向に応じて引き出し軸（X / Y / Z）を自動選択 |
| **ラベル BOX** をドラッグ | ラベルのみ自由移動。寸法線中点との間にリーダー線を表示 |
| **Delete キー** | 選択中の寸法を削除 |

> - 引き出し線は X / Y / Z 軸のいずれかに整列します
> - ピックポイントが寸法平面上にない場合はサポート線を自動表示します

## 7. コメント

ツールバーの **コメントモード** に切り替えて、モデル上の任意の位置にテキストコメントを配置できます。

### コメントの追加

1. ツールバーの「吹き出し」アイコンをクリックしてコメントモードに切り替え
2. モデルの面をクリック → テキスト入力ダイアログが表示
3. コメントを入力して確定 → モデル面上にピンと引き出し線付きでラベルが配置される

### コメントの操作

| 操作 | 動作 |
|------|------|
| **ラベルをドラッグ** | 引き出し線を保持したまま、ラベルを自由移動 |
| **ラベルをダブルクリック** | インライン編集モードに切り替え |
| **Enter / Escape / フォーカス外し** | 編集を確定 |
| 空文字で確定 | コメントを削除 |

## 8. 穴・円筒の自動検出

ツールバーの **穴マーク** アイコンをクリックすると、モデルのメッシュを解析して穴・円筒面を自動検出します。

### 検出結果の見方

| 記号 | 意味 |
|------|------|
| **H1, H2, …** | 穴グループ（同じ径の穴をまとめたもの） |
| **C1, C2, …** | 円筒グループ（外側の円筒面） |

- 同一径の面をグループ化し、記号（H1, H2…）と直径（φ寸法）を 3D アノテーションで表示
- 検出面をグループごとにカラーオーバーレイ
- アノテーションラベルはドラッグで自由に移動可能

### 穴検出パネルの操作

- 穴マークアイコンをクリックするたびに、検出パネル／アノテーションの表示・非表示が切り替わります

## 9. 表示モード

ツールバー右グループで 4 つの表示モードを切り替えられます。

| モード | 用途 |
|--------|------|
| **シェード** | 形状の確認（エッジなし・シンプル） |
| **シェード＋エッジ** | 通常の CAD ビューア風（デフォルト）。エッジを重ねて形状を明確に表示 |
| **陰線** | 見えないラインを隠したワイヤーフレーム表示 |
| **ワイヤーフレーム** | エッジのみを表示。構造確認に便利 |

## 10. デバッグパネル（上級者向け）

**Ctrl + Shift + D** でデバッグパネルを開閉します。色・照明をリアルタイムに調整できます。

### 色設定

| 項目 | 説明 | デフォルト |
|------|------|-----------|
| 背景 | ビューポートの背景色 | `#f0f0f0` |
| モデル | モデルのシェード色 | `#d7d7d7` |
| 鋭角エッジ | 面角度が鋭いエッジの色 | `#111111` |
| 稜線 | 面間の緩い稜線・タンジェント接続部 | `#888888` |
| シルエット | 視点依存の輪郭線 | `#888888` |

### 照明設定

| 項目 | 説明 | デフォルト |
|------|------|-----------|
| 環境光 | 全体的な環境光（HemisphereLight） | 1.0 |
| キーライト | 主照明の強さ（DirectionalLight） | 2.0 |
| フィルライト | 補助照明の強さ（DirectionalLight） | 1.0 |

「Reset」ボタンでデフォルト値に一括リセットできます。

## 11. iFrame 組み込み / postMessage API

BlueMark を iFrame として親アプリに埋め込み、JavaScript の `postMessage` でファイルデータを受け渡すことができます。

### 基本的な組み込み

```html
<iframe
  id="cadViewer"
  src="https://your-deployed-url/"
  style="width:100%; height:600px; border:none;">
</iframe>
```

### URL パラメータ

| パラメータ | 値 | デフォルト | 説明 |
|-----------|-----|-----------|------|
| `fileOpen` | `true` / `false` | `true` | `false` にすると「ファイルを開く」ボタンとドロップゾーンを非表示にする |

```html
<!-- postMessage 専用モード（UIからのファイル選択を非表示） -->
<iframe src="https://your-deployed-url/?fileOpen=false" ...></iframe>
```

### ビューア準備完了の通知

BlueMark は読み込み完了後に親ウィンドウへ次のメッセージを送信します。

```javascript
window.parent.postMessage({ type: 'viewerReady' }, '*')
```

このメッセージを受け取ってから `loadStep` を送ることで、確実にファイルを読み込めます。

### STEP ファイルの送信

```javascript
window.addEventListener('message', (e) => {
  if (e.data?.type === 'viewerReady') {
    // ビューアの準備ができたらファイルを送信
    const response = await fetch('/path/to/model.step')
    const buffer = await response.arrayBuffer()

    document.getElementById('cadViewer').contentWindow.postMessage(
      { type: 'loadStep', buffer, name: 'model.step' },
      '*'
    )
  }
})
```

### postMessage API 一覧

| type | 内容 | 状態 |
|------|------|------|
| `loadStep` | STEP ファイルのロード | 実装済み |
| `setDisplayMode` | 表示モードの切替 | 予定 |
| `setColors` | 色設定の変更 | 予定 |
| `setCamera` | カメラ位置・方向の指定 | 予定 |
| `resetCamera` | カメラをホームに戻す | 予定 |
| `measure` | 計測モードの切替 | 予定 |

### loadStep のパラメータ

| フィールド | 型 | 必須 | 説明 |
|-----------|----|------|------|
| `type` | `string` | 必須 | `"loadStep"` 固定 |
| `buffer` | `ArrayBuffer` | 必須 | STEP ファイルのバイナリデータ |
| `name` | `string` | 任意 | ファイル名（省略時: `"model.stp"`） |
