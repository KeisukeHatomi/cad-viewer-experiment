STEP ファイルを選択すると 3D モデルが表示されます

Connection
src https://cad-viewer-experiment.vercel.app
allow cross-origin-isolated
postMessage API
type: "loadStep"required STEP ファイルをロードして表示
bufferrequired ArrayBuffer STEP ファイルのバイナリデータ
nameoptional string ファイル名（省略時: "model.stp"） 
const res = await fetch('/model.step')
const buffer = await res.arrayBuffer()

frame.contentWindow.postMessage(
  {
    type:   'loadStep',
    buffer: buffer,
    name:   'model.step',
  },
  'https://cad-viewer-experiment.vercel.app'
)
