Features

現場で使える機能を、すべて。

CAD エンジニアが日々の作業で必要とする表示・確認機能をブラウザ上に凝縮。 重いソフトを起動する手間なく、ファイルを開いた瞬間から作業を始められます。

複数の表示モード

Shaded · Shaded-Edges · Hidden-Line · Wireframe の 4 モードを切り替え。形状確認から検図まで用途に合わせて表示を選択できます。

寸法測定

2 点をクリックするだけで 3D 空間上の距離を即時計測。測定値は画面にオーバーレイ表示され、ミリ単位で確認できます。

ピンコメント

モデル上の任意の点にコメントピンを打ち込み。「この穴の公差を確認」「エッジ処理が必要」など、レビュー指摘箇所を 3D 上でマーキング。

TrackballControls

ジンバルロックのない CAD ライクな自由回転。マウスドラッグで任意の方向へスムーズに回転でき、正確な視点操作が可能です。

埋め込み対応

postMessage API で親アプリからの STEP データ受け取りに対応。既存の Web サービスや社内ツールに iframe で組み込めます。

STEP / STP 対応

ISO 10303 準拠の STEP 形式をネイティブサポート。occt-import-js により、業界標準の 3D CAD データをそのままブラウザで読み込みます。

Display Modes

4 つの表示モードで
すべての確認作業に応える。

用途に応じた表示切り替えがワンクリック。意匠確認から部品レビューまで、シーンに最適なビジュアルを選べます。

Shaded-Edges

面のシェーディング＋エッジ強調。もっとも直感的に形状を把握できる標準モード。

Wireframe

すべてのエッジをライン表示。内部構造や隠れた稜線を見通すエンジニア向けモード。

Hidden-Line

手描き図面風の陰線消去表示。検図・設計レビューで構造把握に最適です。

Shaded

エッジなし滑らかシェーディング。表面の意匠確認やプレゼン用途に適したクリーンな表示。

Tech Stack

信頼できる技術基盤。

  • occt-import-js — WebAssembly 版 OpenCASCADE Technology による高精度 STEP パーサー
  • Three.js — WebGL レンダリングによる高品質 3D 描画エンジン
  • TrackballControls — ジンバルロックなし。3 自由度の直感的なマウス操作
  • CSS2DRenderer — 3D 座標に追従するアノテーション・測定ラベルのレンダリング
  • React + Vite — 高速開発・本番ビルドを実現するモダン Web フロントエンド
  • postMessage API — 親アプリからの STEP バイナリ受け渡しに対応。iframe 埋め込みが可能
